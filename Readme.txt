When runniong this project, ensure that GUI.java is the main being executed.


Also, while this project can read in the full Frankenstein.txt document, it requires a lot of RAM (upwards of 12GB),
and will take several minutes to build the trie, during which the gui window will be unresponsive.

This project reads in a text document and constructs a trie, which can then be used for string matching.
